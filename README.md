# Developer Toolbox | Peteware

---

## Overview

**Developer Toolbox** is a versatile Roblox exploit development suite built to help exploit creators and script developers efficiently build, debug, and manage their exploits. It offers a set of essential tools to speed up exploit workflows and enhance testing.

This project is **fully open source**, allowing anyone to access, modify, and contribute to the code. Community collaboration ensures the toolbox stays up-to-date with the latest exploit techniques.

---

## Features

- **Queue on Teleport**  
  Automatically runs scripts when teleporting or server hopping, keeping exploits persistent.

- **Server Hop & Rejoin Buttons**  
  Quickly switch or rejoin servers without losing your exploit context.

- **Client Anti-Kick Toggle**  
  Blocks kick attempts triggered by LocalScripts to keep your exploit active.

- **Instance Scanner**  
  Search game instances by class or property to find targets for exploits.

- **Addons Manager**  
  Add, manage and execute your own scripts from the addons manager.

- **Quick Launch for Popular Tools**  
  Includes buttons for well-known utilities like Infinite Yield, Remote Spy, Dex Explorer, Hydroxide, and Ketamine

- **Executor Info**
  Prints useful executor information such as executor level and executor environment.
